# AS QS Relay

Author: AlphaSys  
Version: 0.1.3  
Status: MVP

## Purpose

Maintains a first-party JSON cookie of tracked query-string touchpoints over time.

## Important Notes

- Reset command: `?asqr_reset=1`
- Independent Analytics hydrate command: `?asqr_from_ia=1`
- JavaScript payload: `window.ASQR_QS_RELAY`
